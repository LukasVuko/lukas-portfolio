My process

  • Create design mockups and divide into appropriate UI components.
  • Architect UI components into React components and style modules (SASS and CSS) - leaving out Javascript functionality.
  • Mobile layout is completed. If required, add basic Javascript functionality like animations, simple click events etc.
  • Media queries are added to match larger viewport sizes.
  • Test code in multiple browsers to ensure cross-browser compatibility.
  • Add core functionality using JS. Things to be considered: local/global state management, routing etc.
  • Finishing touches. Upload project to GitHub where it will ultimately never be seen.
